game ular (DOM)

game ular v1

DESKRIPSI

Game Ular (DOM) adalah game ular sederhana yang dibuat menggunakan HTML, CSS, dan JavaScript. Game ini menggunakan manipulasi DOM untuk menggerakkan ular dan menampilkan makanan.

Pemain mengontrol arah ular menggunakan tombol panah pada keyboard untuk mengambil makanan dan menambah skor.

Project ini dibuat sebagai latihan logika dasar pemrograman seperti percabangan, perulangan, event keyboard, dan manipulasi elemen HTML.

fokus pembelajaran

  • Manipulasi DOM
  • Event keyboard
  • logika gerakan sederhana
  • dasar JavaScript

game ular (canvas)

DESKRIPSI:

Game Ular (Canvas) adalah versi lanjutan dari game ular sederhana yang dibuat menggunakan HTML Canvas dan JavaScript.
